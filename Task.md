# DAB application

## Purpose

The application is a DAB based and ncursed (or cli-) application where the user can:

- scan for stations based on country
- select station to listen to based on the station name
- listen to the selected station using the ui
- listen to the selected station using cli only
- select the output audio device using a parameter (where default is selected)

## Requirements

- Limit third party dependencies
- Low power utilization

## Hardware

Suggestions:
